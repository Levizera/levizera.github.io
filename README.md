# Portfólio de Levi Ferreira

Esta pasta contém a versão estática do portfólio, pronta para o GitHub Pages.
Não é necessário instalar programas nem executar comandos.

## Como publicar

1. Crie um repositório público no GitHub.
2. Envie **todo o conteúdo desta pasta** para a raiz do repositório.
3. No repositório, abra **Settings → Pages**.
4. Em **Build and deployment**, escolha **Deploy from a branch**.
5. Selecione a branch **main**, a pasta **/(root)** e clique em **Save**.
6. Aguarde alguns minutos. O GitHub mostrará o endereço público do site.

O arquivo `.nojekyll` deve permanecer no repositório. Ele evita que o GitHub
altere a estrutura dos arquivos do site.
